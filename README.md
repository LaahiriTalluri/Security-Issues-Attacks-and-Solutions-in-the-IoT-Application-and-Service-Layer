# Security-Issues-Attacks-and-Solutions-in-the-IoT-Application-and-Service-Layer
Research project analyzing security issues, attacks, and defense mechanisms in the IoT application and service layer, focusing on protocols like MQTT, CoAP, and HTTP along with intrusion detection and edge-based security solutions
What This Project Does

This project studies security issues, attacks, and possible solutions in the IoT Application and Service Layer.
The work focuses on understanding how IoT communication protocols such as MQTT, CoAP, and HTTP can be vulnerable to security threats due to their lightweight design and resource-constrained environments.

The project includes:

A literature review of multiple IoT security research papers

Analysis of application layer vulnerabilities

Identification of common IoT attacks

Study of existing security mechanisms and defense techniques

Discussion of future research directions for securing IoT systems

The goal is to provide a clear overview of IoT application layer security challenges and the techniques used to mitigate them.

Repository Contents

This repository contains the research report, presentation, and reference papers used for the study.

Main contents include:

📄 Project Report – Detailed analysis of IoT application layer security

📊 Presentation Slides – Summary of the research work

📚 Research Papers – Papers used for literature review

🖼 Images/Diagrams – Figures used in the report
